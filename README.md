# Open Wicam
Open Wicam, used to be called Wicam, is an open source, open hardware coin size camera. Open-Wicam-Android is the source code repository for its android app.

# Dependencies

- gstreamer for android
- libwebsocket
- miniupnpc
- zlib
- openssl
